# IdleRPG - 2.5D 쿼터뷰 자동전투 RPG

Unity 6 기반 2.5D 쿼터뷰 자동전투 RPG 모바일 게임

## 기술 스택
- **엔진**: Unity 6 LTS
- **캐릭터**: SPUM (Soonsoon Pixel Unit Maker)
- **AI 연동**: mcp-unity (Claude Code)
- **에셋 생성**: ComfyUI + PixelLab MCP
- **타겟**: iOS / Android

## 개발 순서
1. 전투 씬 프로토타입 (게임성 먼저)
2. 전투 깊이 (스킬, 웨이브, 보스)
3. 캐릭터/팀 시스템
4. 메타 시스템 (가챠, 강화, 오프라인 보상)
