# 성급 시스템 + 시너지 + 탈것 + 던전 설계

## 1. 성급 체계 통일

기존 `HeroRarity`(Common/Rare/Epic/Legendary) → `StarGrade`(1~5성) 전환

```csharp
enum StarGrade { Star1 = 1, Star2, Star3, Star4, Star5 }
```

모든 시스템에 공통 적용: 영웅, 스킬, 탈것, 몬스터, 장비

| 대상 | 성급 범위 | 비고 |
|------|-----------|------|
| 영웅 | 1~5성 | 뽑기 시 고정, 클래스: 전사/궁수/마법사/힐러 |
| 스킬 | 1~5성 | 뽑기 시 고정, 동일 스킬 합성 → 레벨업 |
| 탈것 | 1~5성 | 뽑기 시 고정, 영웅 장착 → 패시브 보너스 |
| 몬스터 | 1~2성 일반, 3성 중보스, 4성 보스, 5성 레이드 | 크기 스케일 |
| 장비 | 1~5성 | 기존 rarity 1~5 유지 |

### 뽑기 확률 (영웅/스킬/탈것 공통)

| 성급 | 확률 |
|------|------|
| 1성 | 60% |
| 2성 | 30% |
| 3성 | 9% |
| 4성 | 0.97% |
| 5성 | 0.03% |

- 천장(Pity) 없음
- 무과금 무한 반복 + 과금으로 가속하는 F2P 모델
- 기준: 뽑기권 100개 = 1,000원

## 2. 스킬 시스템 개편

### 스킬 합성 강화
- 같은 스킬 N개 합성 → 스킬 레벨 +1
- 성급은 변하지 않음, 레벨만 상승

### 스킬 시너지 3종

| 타입 | 조건 | 효과 예시 |
|------|------|-----------|
| 조합 시너지 | 특정 스킬 2개 동시 장착 | "화염구 + 독안개" → 폭발독 (추가 DoT) |
| 속성 시너지 | 같은 속성 스킬 N개 장착 | 화염 스킬 2개 → 화염 데미지 +20% |
| 태그 시너지 | 같은 태그 스킬 N개 장착 | "방어" 태그 2개 → 전체 방어력 +15% |

- SkillData에 `element`(속성), `tags`(태그 배열) 필드 추가

## 3. 탈것 시스템

### MountData (ScriptableObject)
- mountName, starGrade, horseSprite (SPUM 경로)
- bonusHp, bonusAtk, bonusDef, bonusMoveSpeed
- 성급 높을수록 보너스 큼

### 동작
- 영웅 1명에 탈것 1개 장착
- 장착 시 CharacterFactory에서 HorseRoot 활성화 + 스프라이트 적용
- UpgradeManager.ApplyAllBonuses에 탈것 보너스 추가

### 뽑기
- 전용 재화: 탈것 소환석
- 단일/10연차, 천장 없음

## 4. 몬스터 성급 + 크기

| 성급 | 역할 | 크기 배율 | 스탯 배율 |
|------|------|-----------|-----------|
| 1성 | 일반몹 | 1.0x | 1.0x |
| 2성 | 일반몹 (강화) | 1.0x | 1.3x |
| 3성 | 중간보스 | 1.4x | 2.0x |
| 4성 | 에리어 보스 | 1.7x | 3.5x |
| 5성 | 레이드/특수 보스 | 2.0x | 6.0x |

- CharacterFactory에서 starGrade에 따라 transform.localScale 조정
- 스테이지 진행마다 기본 스탯에 계수 누적

## 5. 던전 시스템

### 3종 던전
- 영웅 던전 → 보석
- 탈것 던전 → 탈것 소환석
- 스킬 던전 → 스킬 주문서

### 진행 규칙
- 1~100단계, 순차 해금 (N단계 클리어 → N+1 입장 가능)
- 기존 웨이브 전투와 동일 방식 (제한 시간 내 처치)
- 점수 = 처치수 + 속도 보너스

### 보상
- 클리어 시 단계에 비례한 재화 지급
- 랭킹 시스템: 점수 기반
- 10만점 이상이면 랭킹 밖이어도 보상 수령 가능

## 6. 신규 재화

| 재화 | 용도 | 획득처 |
|------|------|--------|
| 보석 (기존) | 영웅 뽑기 | 영웅 던전, 일일미션, 업적 |
| 탈것 소환석 (신규) | 탈것 뽑기 | 탈것 던전 |
| 스킬 주문서 (신규) | 스킬 뽑기 | 스킬 던전 |
